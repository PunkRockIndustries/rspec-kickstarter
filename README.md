# RspecKickstarter

RSpec kickstarter generates spec files for existing code.

## Installation

    gem install rspec-kickstarter

## Usage

    rspec-kickstarter ./app
    rspec-kickstarter ./lib
    rspec-kickstarter ./lib/yourapp/util.rb
